Welcome to GCI Checkers Page

GCI Checkers is a checkers game created in html, using canvas and javascript.

How to play

You must move a piece every turn, if you can't move anything, you must surrender using the "Surrender" button. To move a piece, you need to select it and the press the "Move" button. If you pressed the "Move" button, there will appear the options marked with a blue squares where you can move. In this stage, if you select anything else other then the blue squares, you will deselect the current piece. If you selected a blue square, you must move that piece until you can't move it any more. After that you can give the turn to the other player using the button "Give Turn".
